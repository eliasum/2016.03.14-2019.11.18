Начало работы

При первом открытии начальной страницы http://klada.nccp-eng.ru/ в кладовке КБА отсутствует оборудование, о чём говорит сообщение "По запросу ничего не найдено!":

 

 

Страница "Добавить наименование" доступна только операторам кладовки КБА, остальным работникам КБА информация по оборудованию доступна только в режиме "для чтения". Чтобы добавить новое оборудование, нажмите "Добавить наименование":

 

 

Откроется окно добавления нового наименования. Впишите в поля соответствующие значения примерно так, как показано ниже и нажмите кнопку "Добавить":

 

 

После этого нужно подтвердить добавление нового оборудования, для чего появится соответствующее сообщение:

 

 

Если нажать "ОК" и всё введено правильно, новое оборудование добавится, а так же будет выведено сообщение "Наименование успешно добавлено! Оприходовано единиц: 1000.":

 

 

Если нажать "Отмена", то добавление нового оборудования будет отменено и выведено соответствующее сообщение. Чтобы вернуться на начальную страницу, нужно нажать "Вернуться на начальную страницу":

 

 

Вернувшись на начальную страницу, вы увидите сформированную таблицу с новым наименованием:
