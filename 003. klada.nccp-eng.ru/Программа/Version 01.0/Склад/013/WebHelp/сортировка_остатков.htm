При запуске начальной страницы сортировка оборудования выставлена по умолчанию (в порядке ввода), а порядок сортировки - прямой:

 

 

 Оборудование можно сортировать как отдельно по наименованию, единице измерения, производителю, коду, дате, так и сразу по нескольким полям одновременно. Например, чтобы выбрать сортировку по единице измерения, нужно выбрать галочку напротив "Единицы измерения" и нажать кнопку "Отсортировать":

 

 

Обратите внимание, что галочка "По умолчанию" была автоматически снята.
